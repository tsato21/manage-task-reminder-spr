About this Project

This project introduces an advanced Google Apps Script to automate task management in Google Sheets, offering a seamless and customizable experience.

  • Efficiently manage tasks with automated reminders.
  • Seamlessly integrate with Google Sheets functionalities.
  • Customizable to fit a variety of use-cases.

Prerequisites

  • A Google account with access to Google Sheets.
  • A foundational understanding of Google Sheets.

Setup

  1. Access Your Google Sheet: Begin by opening the Sample Google Sheet. For your reference, you can find sample Google Documents for reminders from HERE(This will be explained in the later section).
  2. Copy the Sheet: Make a copy of the Google Sheet to create your personalized version.
  3. Authorization: Navigate to the 'Custom Menu' and click the 'Conduct Authorization' button to start the Google Apps Script authorization process.
    Figure: GAS Authorization
  4. Return to Original Format: Navigate to the 'Custom Menu' and click the 'Return to Original Format' button to make the Spreadsheet back to the original format (delete sample task sheets/ clear contents in index sheets/ reset pre-defined sample information/ delete sample triggers).
  5. Customize Constants (Optional): Visit the Apps Script editor to modify constants in `variables.gs` as per your requirements.
    Figure: Customizing Constants

Project Usage

  1. Initial Configuration: Initiate your project setup by navigating to `Custom Menu` > `SETTINGS` > `Set Pre-Defined Info` to tailor the system to your operational needs.
    • General Settings(Mandatory): Configuration on information for index sheets and editor of this spreadsheet.
    • General Reminder Settings: Fundamental for establishing email reminders and configuring URLs of reminders to display target reminders.
    • Staff-Specific Reminder Settings(Optional): Optionally, you can make settings for reminders of individual staff reminders.
    • Google Sheet and Doc Integration(Mandatory): Assign a trigger for monitoring task completion statuses in Google Doc and updating completion filed in Google Sheet.
  2. Task Sheet Generation: Effortlessly create task sheets by selecting `Custom Menu` > `TASK SHEETS` > `Create a New Task Sheet`, designed to support a structured format for task assignment.
    • Delegate tasks seamlessly to team members using the in-built dropdown functionality within the task sheet.
    • To alter the task sheet's access permissions, proceed to `Custom Menu` > `TASK SHEETS` > `Modify Editors of the Current Sheet`.
  3. Reminders Testing: Test the reminder system via `Custom Menu` > `TEST`.
    • Ensure Google Docs are established for each reminder type prior to utilization.
    • If the system works as expected, you will receive an email notification with the link to the Google Doc.
  4. Status Synchronization: Input "C" in "Complete" column for tasks in Google Docs, which is synchronized with your Google Sheets.
    • You can test this synchronization by `Custom Menu` > `TEST` >`Update Completion Status`.
    • Trigger setting in advance is required in `SETTING`.
  5. Tools to Organize Sheet Info: Optimize your sheet organization by using `Custom Menu` > `Update Index Sheets and Sort Task Sheets`.
    • This sorts task sheets by date and update index sheets.
    • Add the pre-set completion flag to the sheet with all tasks completed. This makes the sheet information automatically included in the `completed-task-index`.(*Set the flag in "Setting" in advance.)
