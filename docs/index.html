About this Project

This project involves a Google Apps Script designed to streamline and automate various tasks within Google Sheets.

  • Automated task management and reminders.
  • Seamless integration with Google Sheets functionalities.
  • Customizable scripts for diverse use-cases.

Prerequisites

  • A Google account with access to Google Sheets.
  • Basic understanding of Google Sheets and Google Apps Script.
  • Ability to follow step-by-step setup instructions.

Setup

  1. Open Your Google Sheet: Access Sample Google Sheet.
  2. Copy the Google Sheet to create your version.
  3. Conduct GAS Authorization: Go to `Custom Menu` in the menu bar and click `BEFORE FIRST USE` > `Conduct Authorization`. This will redirect you to the authorization page for Google Apps Script.
  4. Return to Original Format.: Go to `Custom Menu` in the menu bar and click `BEFORE FIRST USE` > `Return to Original Format`. This deletes all sample task sheets and clears the contents in the index sheets. Also, it resets information and triggers pre-defined as a sample. This makes the Google Sheet ready for your use.
  5. Customize Constant Variables for Built-in Functions (Optional): Go to the Apps Script editor and adjust the constants in `variables.gs` to suit your needs. This is optional and without customization, this project will still work.

Usage

  1. Task Management: Leverage the script to efficiently manage and track tasks within your Google Sheet.
  2. Automated Reminders: Configure automated reminders to efficiently manage task deadlines and events.

Setting Up the Project

Follow these steps to set up and start using the project:

  1. Access SETTINGS > Set Pre-Defined Info from the custom menu to configure necessary information.
    • 【General Setting】: Mandatory to configure staff information.
    • 【General Reminder Settings】: Set emails for general reminders, the Google Doc URL for displaying reminders, and triggers for reminders.
    • 【Staff-Based Reminder Settings】: Optional, for individual staff reminders. Prior setup of a Google Doc for each staff is required.
    • 【Others】: Set triggers for updating task completion status.
  2. Create task sheets and enter to-do items.
    • Create new task sheets via Custom Menu > TASK SHEETS > Create a New Task Sheet.
    • Assign tasks to specific staff members using the dropdown in the task sheet.
    • Optionally, test reminder functionalities via Custom Menu > REMINDER.
  3. Update completion status for tasks.

    Mark tasks as complete in the reminder Google Docs to reflect the changes automatically in the Google Spreadsheet.

  4. Manage Task Sheets and Index Sheets.

    Once tasks are completed, rename the task sheet with "Fin)" and update index sheets via Custom Menu > Update Index Sheets.
