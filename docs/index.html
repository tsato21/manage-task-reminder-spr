About this Project

This project introduces a Google Apps Script designed to optimize task management and automation within Google Sheets.

  • Automated task management and reminders for improved productivity.
  • Seamless integration with Google Sheets functionalities.
  • Customizable scripts to cater to a variety of use-cases.

Prerequisites

  • Access to Google Sheets with a Google account.
  • Familiarity with Google Sheets and basic Google Apps Script usage.
  • Willingness to engage with detailed setup instructions.

Setup

  1. Open Your Google Sheet: Begin with the Sample Google Sheet.
  2. Copy the sheet to create your personalized version.
  3. Conduct GAS Authorization: From the `Custom Menu`, select `BEFORE FIRST USE` > `Conduct Authorization` for Google Apps Script authorization.
  4. Return to Original Format: To prepare your sheet for use, navigate to `Custom Menu` > `BEFORE FIRST USE` > `Return to Original Format`.
  5. Customize Constant Variables (Optional): Adjust the constants in `variables.gs` in the Apps Script editor to suit your needs.

Usage

Here's how you can make the most out of this project:

  1. Set Pre-Defined Information: Configure necessary settings through `Custom Menu` > `SETTINGS` > `Set Pre-Defined Info`.
    • 【General Setting】: Mandatory for configuring staff information.
    • 【General Reminder Settings】: Essential for setting up email reminders and Google Doc URLs for displaying reminders.
    • 【Staff-Based Reminder Settings】: Optional, for individual staff reminders.
    • 【Others】: Configure triggers for task completion status updates.

    *Reset or update pre-defined information as needed.

  2. Create task sheets and enter to-do items.
    • Create new task sheets via `Custom Menu` > `TASK SHEETS` > `Create a New Task Sheet`. This enables the sheet to be created with designated format.
    • Assign tasks to specific staff members using the dropdown in the task sheet.

      • If you need to modify editors of the sheet, access `Custom Menu` > `TASK SHEETS` > `Modify Editors of the Current Sheet`.

  3. Test reminders.
    • Access `Custom Menu` > `REMINDERS` > `General Reminders` to test general reminders.
    • Access `Custom Menu` > `REMINDERS` > `Staff-Based Reminders` to test staff-based reminders.

      • *Before using this feature, create Google Docs to display each type of reminders.

  4. Update completion status for tasks.

    Input "C" in the complete cell for completed tasks in the reminder Google Docs to reflect the changes automatically in the Google Spreadsheet.

  5. Update Index Sheets and Sort Task Sheets.

    Use `Custom Menu` > `Update Index Sheets and Sort Task Sheets` to update index sheets and sort all of the task sheets by the date ascending.

    *Once all tasks in a sheet are completed, rename the sheet with "Fin)" to list it in `completed-task-index`.
